# Datos Personales Comprometidos

Tipos de datos afectados:

- **Correos electrónicos** → datos personales de empleados.
- **Información médica** → datos sensibles protegidos por Ley 19.628.
- **Guiones inéditos** → datos corporativos estratégicos.

Derechos ARCO vulnerados:

- **Acceso** (Art. 12 Ley 19.628).
- **Rectificación** (Art. 13).
- **Cancelación** (Art. 14).
- **Oposición** (Art. 15).

El ataque vulneró la privacidad y los derechos fundamentales de los trabajadores.
